FORGE - 0.0.1 BETA

This is Forge, a managed UI application for running LLMs locally on your desktop.  Enabling both experimenting with new LLMs locally, but keep corporate management policies e.g for privacy, control etc.

- Windows x64 support (Mac and Linux coming ..)
- Llama 3.1, OpenAI, Qwen 2.5, Mistral build in support
- Based on LlamaCpp backend
- Support additional models - see llmdefs.json file

Just Install and run!
